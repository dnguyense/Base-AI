# Cursor IDE Rules Configuration

📋 **Base Rules Reference**
Tất cả các quy tắc cơ bản được quản lý tập trung tại:
- .appdexer/rules/global-rules.md - Quy tắc chung cho tất cả projects
- .appdexer/rules/project-specific-rules.md - Quy tắc riêng cho project này

⚠️ **Lưu ý:** Vui lòng tham khảo các file rules chính thức ở trên thay vì nội dung trong file này.

## Cursor-Specific Instructions
- Sử dụng rules từ .appdexer/rules/ làm nguồn chính thức
- File này chỉ chứa các cấu hình đặc thù cho Cursor IDE
